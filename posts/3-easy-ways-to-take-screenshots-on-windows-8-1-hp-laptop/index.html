First, you can press the “Print Screen” (PrtSc) key on your keyboard to take a screenshot of your entire screen. The screenshot will be saved to your clipboard, which you can then paste into any image editing program (such as MS Paint or Adobe Photoshop).

Second, you can press the “Alt + Print Screen” (Alt + PrtSc) keys to take a screenshot of the currently active window. Again, the screenshot will be saved to your clipboard, ready to be pasted into any image editing program.

Third, you can use the Windows 8.1 Snipping Tool. This helpful tool allows you to take a screenshot of any area of your screen, and even allows you to draw, highlight, or add text to your screenshot before saving it.

1. Adjust Settings on Windows 8.1 HP Laptop#

Taking screenshots on a HP laptop running Windows 8.1 is a breeze once you know how. Here are three easy methods that use the built-in tools of Windows 8.1, so you can get to capturing your screen quickly.

First, use the Print Screen (PrtScr) key. Pressing this key will save an image of your entire screen to the clipboard. You can then open a program such as Paint and paste the image. If you are using multiple monitors, you can also use the Alt + PrtScr keys to capture a single monitor at a time.

Second, you can use the Snipping Tool, which is part of the Windows Accessories folder. With this tool you can capture portions of the screen or the entire screen as an image. You can also delay the capture by up to five seconds, if there are elements of the screen you would like to include that don’t appear immediately.

Lastly, you can use the Windows 8.1 Game Bar. To access this open the charms bar (Windows key + C) and type “Game Bar” in the settings search. Once the bar is enabled, you can capture a screenshot by pressing Windows + PrtScr. The image will be automatically saved to the “Videos” folder within “Pictures” on your PC.
